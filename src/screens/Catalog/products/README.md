# Products Module

Handles all product-related functionality including listing, filtering, adding, editing, and managing products.

## Files

- **ProductsScreen.tsx**: Main products list screen with filters and sorting
- **AddProductScreen.tsx**: Form for adding/editing products
- **index.ts**: Module exports

## Features

- Product listing with search
- Advanced filtering (Inventory, Discount, Price Range)
- Multiple sort options
- Add/Edit/Delete products
- Image upload (up to 6 images)
- Add products to collections
- Mark products as in/out of stock
